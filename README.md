# challenges

Repositorio de proyectos para hackathons y competencias.

## Proyectos

### 1. [INTI Space](./Space%20Apps/INTI%20Space/) — NASA Space Apps Challenge 2024

Portal educativo sobre Objetivos de Desarrollo Sostenible (ODS) usando datos satelitales de la NASA.

- **Stack:** HTML5, CSS3, JavaScript, Bootstrap 5, jQuery
- **Build:** Node.js script con partials HTML
- **Deploy:** Vercel → https://inti-space.vercel.app/

```bash
cd "Space Apps/INTI Space"
npm run build    # Genera sitio estático en dist/
npm run dev      # Build + servidor local
```

### 2. [Stacks / Cusco Nakamoto](./Stacks/Cusco%20Nakamoto%20Workshop/) — Blockchain Workshop

Smart contracts en Clarity para la blockchain Stacks.

- **Stack:** Clarity, Clarinet, TypeScript, Vitest
- **Contracts:** `hello_world.clar`, `transaccion.clar`
